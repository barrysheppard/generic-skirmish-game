# Injuries and Death

Each fighter taken down in battle must roll on an injury table.

Most Fighters have a simple table, roll a single D6

|D6|Injury Effect|
|:--:|:--|
|1-2|The Fighter has died and is removed.|
|3-4|The Fighter has been injured and misses the next game.|
|5-6|The Fighter has survived without injuries.|

**Leaders** and **Champions** have a more detailed table roll two D6 in order D66. In cases where there is a long term effect in the Injury Effect, this is treated as a passive Ability of the same name as the injury and can be added to the Fighter profile as a Trait.

| <div style={{width: '55px'}}>D66</div> | <div style={{width: '125px'}}>Injury Name</div> | Injury Effect |
| :--- | :--- | :--- |
| **11-16** | **Death** | The Fighter has died and is removed from the Roster along with their Gear. |
| **21-23** | <span style={{color: 'red', fontSize: '1.2em'}}>▼</span> **Gut Wound** | **[passive]** This Fighter's wound characteristic is reduced by 2. This injury can be taken multiple times. Decrease this Fighter's cost by 5. |
| **24-26** | <span style={{color: 'red', fontSize: '1.2em'}}>▼</span> **Cracked Rib** | **[passive]** This Fighter can only take one move action per turn. If this injury is rolled a second time, treat it as a Flesh Wound. Decrease this Fighter's cost by 5. |
| **31-33** | <span style={{color: 'red', fontSize: '1.2em'}}>▼</span> **Eye Injury** | **[passive]** This Fighter's Ballistic Skill is reduced by 1. This injury can be taken multiple times (to a minimum of 2 BS). Decrease this Fighter's cost by 5. |
| **34-36** | <span style={{color: 'red', fontSize: '1.2em'}}>▼</span> **Broken Arm** | **[passive]** This Fighter's Weapon Skill is reduced by 1. This injury can be taken multiple times (to a minimum of 2 WS). Decrease this Fighter's cost by 5. |
| **41-43** | <span style={{color: 'red', fontSize: '1.2em'}}>▼</span> **Leg Fracture** | **[passive]** Subtract 1 from the Move characteristic of this fighter. This injury can be taken multiple times (to a minimum of 2 Move). Decrease this Fighter's cost by 5. |
| **44-46** | <span style={{color: 'red', fontSize: '1.2em'}}>▼</span> **Head Trauma** | **[passive]** This fighter cannot use abilities in the first battle round, and thereafter can use only one ability per round. If this injury is rolled a second time, treat it as a Flesh Wound. Decrease this Fighter's cost by 5. |
| **51-65** | **Flesh Wound** | This Fighter suffers no effects. This is not considered an injury. |
| **66** | **Lucky Escape** | Survival against the odds, the Fighter gains 1 experience point and receives no injury. |

A Fighter can have at most 3 <span style={{color: 'red', fontSize: '1.2em'}}>▼</span> injuries, if they would take a 4th injury with a lasting negative effect (eg. 21-26 <span style={{color: 'red', fontSize: '1.2em'}}>▼</span>) treat it as Death instead.

:::info Philosophy
A discount of 5 definitely isn't enough, often the actual impact will be 10 or even 20. Rather than try get a perfect number, we've given the lowest discount option so having this injury will never be a benefit.
:::
