# 8.1 Campaign Roster

Each player maintains a persistent **Roster** representing their overall force. The Roster contains all the available Fighters which player has purchased,

The **Roster** tracks:
- Profiles of all Fighters purchased for the Roster
- Gear each Fighter has
- Any Advancements a Fighter may have taken
- Unspent Points

Before each Mission, you select fighters from your **Roster** to build your Warband.

At the end of each Mission, update the **Roster** to track changes to Fighters and to include new additions.

As Fighters are distinct, they should be given a name when they are added to the **Roster**. This will help track unique Fighters as they grow and develop.

The **Roster** must always have at least one **Leader**. If your **Roster** has no **Leader**, you can select a Fighter with the **Champion** trait. They lose the **Champion** trait and gain the **Leader** trait instead. If no **Champion** is available you can select any Fighter and give them the **Leader** trait. Alternatively, if funds are available the **Roster** can simply purchase a new **Leader**.
