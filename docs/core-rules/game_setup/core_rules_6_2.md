# Battlefield Setup

## 6.2 Battlefield Preparation

- The default board is a 22" by 30" play area. Deployment typically occurs in the centre and sides of the board edges, so keeping these areas clear for deployment would be good.

:::info Philosophy
Philosophy: Board size only really matters for deployment maps. We’ve stayed with the Warcry/Killteam sized board as it’s the perfect size for a game store where it roughly is the size of two Magic the Gathering mats. If a game store can host a MtG event, then it can host a GSG game.
:::

- Determine which long side of the board is 'North', this will be used to align the deployment in a later step.
- Board and terrain can be prepared in a way that makes for a fun looking and enjoyable play experience. This should be done before the mission is selected, so no player will know where their fighters will be deploying. This should encourage players to create a well balanced board, and if one side does end up being better then it will be random who gets the advantage from the deployment.
- The amount of terrain used will change how the game is played. An empty board will make ranged attacks more effective. Terrain with extra levels will let Fighters climb and jump for more variety. The reality is, you can have a fun game on any sort of board whether it's a Warhammer World visual delight or a kitchen table with cans for terrain.
- Consider any Special Rules that apply for the terrain, this can be noted by assigning Traits (and by doing so Abilities) to terrain pieces.
