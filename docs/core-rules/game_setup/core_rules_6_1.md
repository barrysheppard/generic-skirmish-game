# Mission Overview

##6.1 Mission Setup

In the default approach, the **Deployment**, **Victory Condition**, and **Twist** details are randomly selected from a limited pool of options. This approach gives good variety and replayability but may result in some combinations which favour one Warband or another.

An alternative approach, recommended for matched play, is to use missions where the **Deployment**, **Victory**, and **Twist** are already combined and randomly selected from a number of missions. This approach makes sure the combination works well together but does limit variety.

The recommended order for default games is as follows:
- Prepare the game board with terrain.
- Determine the Victory condition.
- Select Warbands for the Mission.
- Determine the deployment.
- Determine the twist.
