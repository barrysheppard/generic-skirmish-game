# Playing the Game

## **7.2 Mission rounds** {#7.2-battle-rounds}

Each Mission round consists of four phases:
- The Initiative Phase
- The Reserve Phase
- The Combat Phase
- End Phase


### 7.2.1 The Initiative Phase {#7.2.1-initiative-phase}
   - Each player rolls 8 Ability dice (D6).
   - Players groups their Ability dice into groups of the same number (i.e. all the 2s together, etc). These groups are referred to as [double]s, [triple]s and [quad]s based on the number of dice in each group. Hold onto these dice for the turn and don't change their values. Groups cannot be split at this point and groups of 4 or more dice are considered to be [quad]s.
   - The player with the highest number of singles (groups of 1 dice) ‘has the initiative’, if this is equal then it is randomly decided with a dice.
   - Starting with the player with the initiative, each player gains a Wild dice to their pool, any Wild dice they have can be spent to change a [single] to a [double], a [double] to a [triple], or a [triple] to a [quad]. Each group can only have one Wild dice, so you cannot upgrade a [double] to a [quad]. Unused Wild dice are kept and can be used in later turns.
   - Determine who 'has the initiative' again to account for any changes in Ability dice. The player who ‘has the initiative’ then chooses who will be the ‘first player’ for the turn.

:::info Philosophy
We're rolling 8 dice instead of 6 which is 33% better! The thinking here is to increase the number of abilities used per turn. We will also allow splitting of groups once we get into the Combat phase, so if you have a set of 5 and use it for a [triple] you can later use it for a [double]. This is to account for the larger pool.
:::

### 7.2.2 Reserve Phase {#7.2.2-reserve-phase}
   - Battle groups arriving this turn are placed on the board. The first player deploys all their fighters followed by the second Player.
   - Each fighter must deploy wholly within 3 inches of their deployment point.
   - Note that when the deployment point is a battlefield edge, fighters from the same battle group do not need to be deployed next to each other.

### 7.2.3. Combat Phase {#7.2.3-combat-phase}
   - Players alternate activating fighters that have not yet activated, starting with the first player.
   - An activated fighter takes actions until unable to do so, typically this will involve 2 standard actions and possibly some abilities.
   - After a fighter is activated they are given an activated token. 

### 7.2.4. End Phase {#7.2.4-end-phase}
   - Check for end of round effects.
   - Count victory points.
   - Reset tokens and clear Ability dice.
   - If this is the end of the fourth round, the game ends and a victory is declared.
