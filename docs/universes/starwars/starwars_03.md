# **Character Trait Abilities**

This section defines character-based traits across several skill categories. Each trait grants a unique ability of the same name that the fighter can use in battle. Some fighters like the Captain and Champions gain access to Skills at the start, other fighters may gain access to skills as they advance through a campaign.

When a fighter gains a skill, they pick from one of the available skill groups. If this is the first rank in that skill group they gain the rank 2 skill. If they subsequently pick that skill group again for a skill, they get to pick from one of the two skills in rank 2\.

Note \- This section is in early draft stages.

---

| Skill Group | Rank 1 | Rank 2 |
| :---- | :---- | :---- |
| Shooting  | Quick Shot | Pistolier or Eagle Eye |
| Magic  | Curse | Teleport or Heal |
| Melee  | Strike to Injure | Combat Master or Weapons Training |
| Weapon Skill  | Might Blow | Pit Fighter or Resilient |
| Agility  | Leap | Sprint or Acrobat |
| Command  | Voice of Command | Inspirational or Iron Will |

## **1\. Shooting Skills**

### 

| Name | Cost | Effect |
| :---- | :---: | :---- |
| **Quick Shot** | \[double\] | This fighter may make a bonus ranged attack action with one fewer attack die. This action must target a visible enemy within half the weapon’s maximum range. |
| **Pistolier** | \[double\] | For this fighter's next attack they may target two different enemies within 6" when using a paired set of ranged weapons, reducing the dual weapon’s attacks by 1, and rolling separately for each target. |
| **Eagle Eye** | \[double\] | Add 5" to the maximum range of this fighter’s next ranged attack action this turn. |
| **Trick Shooter** | \[double\] | For the next ranged attack made by this fighter, when determining line of sight it does not have to be a straight line but that new line is also used for checking range. |
| **Aimed Shot** | \[triple\] | If this fighter has not moved this round, they deal \+1 damage on their next ranged attack and cannot take move actions for the remainder of this turn. |
| **Marksman** | \[double\] |  |

## **2\. Magic Skills**

| Name | Cost | Effect |
| :---- | :---- | :---- |
| **Teleport** | \[quad\] | Remove this fighter from the battlefield and set them up anywhere on the battlefield floor more than 5" from any enemy fighters. |
| **Curse** | \[triple\] | Choose one visible enemy within 6". Allocate D3 damage and reduce their Defense by 1 until the end of the round. |
| **Heal** | \[double\] | Choose a friendly fighter within 3". Remove damage points from them equal to the value of this ability. |
| **Arcane Shield** | \[double\] |  |
| **Foresight** | \[double\] | Change a saved set of Ability dice to be \+1 dice value up to a maximum of 6\. |
| **Mana Surge** | \[triple\] |  |

## **3\. Melee Skills**

| Name | Cost | Effect |
| :---- | :---- | :---- |
| **Strike to Injure** | \[triple\] | Remove this fighter from the battlefield and set them up anywhere on the battlefield floor more than 5" from any enemy fighters. |
| **Combat Master** | \[quad\] | This fighter may make a melee attack against each visible enemy within 1", rolling separately for each. |
| **Weapons training** | \[double\] | This fighter may use two different melee weapon profiles in one activation. |
| **Wall of Steel** | \[double\] | Reaction: When damaged by a melee attack, roll a dice. On a 5+, the attacker suffers 2 damage. |
| **Victory Rush** | \[double\] | A fighter can only use this ability if an enemy fighter has been taken down by an attack action made by them this activation. Remove a number of damage points allocated to this fighter up to half the value of this ability (rounded up). |
| **Knife-Fighter** | \[double\] | Reaction: This fighter may make a bonus melee attack with a throwing knife or dagger if an enemy ends their movement within 3". |

| :---- | :---- | :---- |
| **Mighty Blow** | \[triple\] | This fighter’s next melee attack deals triple damage on a critical hit. |
| **Pit Fighter** | \[double\] | If this fighter is within 1" of two or more enemies, add \+1 to their Attacks characteristic for this activation. |

| **Unstoppable Charge** | \[triple\] | After a move action that ends within 1" of an enemy, deal 2 damage to that enemy and make a free melee attack. |

## **5\. Agility Skills**

### 

| Name | Cost | Effect |
| :---- | :---- | :---- |
| **Leap** | \[double\] | This fighter may move horizontally over gaps up to 3" wide during this activation without needing to jump. |
| **Sprint** | \[triple\] | This fighter may make two consecutive move actions as a single activation. |
| **Acrobat** | \[double\] | Ignore vertical distance when this fighter climbs or leaps this turn. |
| **Lightning Reflexes** | \[double\] | Reaction: Use when targeted by a melee attack. On a 5+, avoid all damage from the attack. |
| **Step Aside** | \[double\] | Reaction: Use when an enemy finishes a move action within 1". This fighter may immediately make a free disengage action. |
| **Nimble** | \[double\] | Use when this fighter is hit by a ranged attack, but before damage is assigned. Subtract 1 from the attacker’s hits (but not crits) for this action. |

| Name | Cost | Effect |
| :---- | :---- | :---- |

| **Inspirational** | \[double\] |  |
| **Iron Will** | \[double\] | Until the end of the battle round, add 2 to the Defense characteristic of friendly fighters with a Shield while they are within 3" of this fighter. |
| **Mentor** | \[double\] | Pick a visible friendly fighter within 6" of this fighter that has not activated yet this battle round. You can activate that fighter immediately after this fighter's activation ends. |

## **Stealth Skills**

| Name | Cost | Effect |
| :---- | :---- | :---- |
| **Fade into Shadows** | \[double\] | This fighter gains a Stealth token. |
| **Shadow Step** | \[double\] | Choose a point within 6” of this fighter that is not visible to any enemy fighter. Remove this fighter from the battlefield and set it up at that point. |
| **Smoke Bomb** | \[double\] | Friendly fighters within 6" gain a Stealth token if they are not visible to any enemy fighters at the time this ability is used. |
| **Assassin’s Mark** | \[triple\] | Pick a visible enemy fighter within 6". That fighter becomes Marked. While within 1" of a friendly fighter with a Stealth token, reduce its Toughness by 1\. |
| **Silent Ambush** | \[triple\] | This fighter can make a bonus move if it has a Stealth token and is not visible to any enemy at the start of its activation.|
| **Vanish** | \[Quad\] | Remove this fighter from the battlefield. At the start of the next battle round, set it up anywhere on the battlefield more than 6" from all enemy fighters. It gains a Stealth token. |
