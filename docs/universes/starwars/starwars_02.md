# Weapons

Unlike with other variants of GSG, by default weapons are not purchased seperately from the fighters. Instead troopers come with standard equipment when purchased which match with their background. Named also will have weapons and gear already assigned to them. Below is a list of weapons and their costs if you re making your custom fighters.

## Weapon Traits

| Name | Cost | Effect |
| :---- | ----- | :---- |
| **Accurate** | \[double\] | Increase the Weapon Skill of attacks with this weapon this turn by 1\. |
| **Armour Piercing** | \[double\] | For the next attack with this weapon this activation, if the target has a Def of 5 or greater, reduce the Def of the target by 2\. |
| **Beskar** | \[passive\] | For the purposes of the Lightsaber trait, this weapon is considered a Lightsaber. |
| **Blast (X)** | \[double\] | For the next attack with this weapon this activation, all fighters within X” of the target take 2 damage if the attack rolls at least one hit or crit. |
| **Convertible** | \[double\] | At the start of the game select one of the three weapon configurations, that is considered the equipped weapon. Use this double to switch the weapon to a different mode. |
| **Indirect Fire** | \[passive\] | Rather than selecting a fighter, this weapon can select a point on the map as the target, if they do so the DEF of the target is 5\. |
| **Lightsaber** | \[passive\] | For the next attack with this weapon this activation, if the target has a Def of 5 or greater, reduce the Def of the target by 2\. Reaction abilities cannot be triggered against an attack with this weapon unless the reaction comes from a weapon with the Lightsaber trait. |
| **Limited (X)** | \[passive\] | This weapon can only be used X times per game. |
| **Parry** | \[double\] | Reaction: After this fighter has been hit by an attack action but before damage is calculated, change one crit into a hit or one hit into a miss. |
| **Quick Draw** | \[double\] | Before attacking with this weapon, this fighter can take a 2 inch move (but not disengage). |
| **Rapid Fire** | \[double\] | For the rest of the turn attacks with this weapon gain \+1 attack dice. |
| **Reload** | \[passive\] | This weapon may only be fired once per turn. |
| **Scope** | \[passive\] | This weapon gains \+1 Weapon Skill if the shooter has not moved this turn (they can still move after shooting). |
| **Silent** | \[passive\] | Attacks made with this weapon do not break Stealth. Fighters cannot react to this attack. |
| **Stun** | \[passive\] | On a critical hit, the target loses an action if possible (gain a wait token or change a wait token to an activated token). |
| **Suppressive Fire** | \[double\] | After making an attack, give the target of that attack a Suppressive Fire token, this token gives the target \-1 Attack to all their attacks and \-1" and is removed at the end of the target’s next activation. |
