# **Extra Gear** {#extra-gear}

Sometimes when building a squad you’ll find you have some extra points left over. The gear items below cost a few credits and give your fighter an extra ability they can use. 

| Name | Cost | Effect | Cr |
| :---- | ----- | :---- | :---: |
| **Comlink** | \[passive\] | This fighter may use an ability with a range, ignoring the range, if the target is a friendly fighter with a Comlink. | 5 |
| **Macrobinoculars** | \[double\] | Reveal one hidden or Stealthed enemy fighter within 12" and line of sight. That fighter loses any Stealth token and cannot gain one until the end of round. | 10 |
| **Datapad** | \[passive\] | This fighter gains the Computers (2) skill and an extra dice when interacting with computer terminals.| 5 |
| **Bacta Patch** | \[double\] | Heal this fighter or a friendly fighter within 1” for 4 wounds. Droids cannot benefit from this effect. This ability can only be used once per game. | 20 |
| **Field Rations**	 | \[double\] | Remove all negative tokens from this fighter (Suppressed, Weakened, etc.). Droids cannot benefit from this effect. This ability can only be used once per game. | 5 |
| **Emergency Beacon** | \[triple\] | Once per game, place a reinforcement point within 3” of this model. Friendly fighters may deploy from this point rather than their normal deployment spot. This ability can only be used once per game. | 20 |
| **Climbing Line**	 | \[passive\] | This fighter may climb vertical surfaces up to 6” high without spending extra movement. | 10 |
| **Vox Scrambler** | \[double\] | Fighters within 6” cannot issue or benefit from abilities involving communication this round. | 10 |
| **Rebreather Mask**	 | \[passive\] | Immune to gas effects and suffocation hazards.| 5 |
| **Smoke Canister** | \[double\] | Create a 3" diameter smoke area within 6" that blocks line of sight and grants cover. Disperses at the end of round. This ability can only be used once per game. | 15 |
| **Repair Kit** | \[passive\] | This fighter gains the Mechanic (2) skill and an extra dice when interacting trying to repair. | 5 |
