# Introduction
This setting pack goes back to the original edition of Warcry and the six warbands battling it out in the Eightpoints.

Fighters match the ones in those original warbands but this pack is designed with the intention of supporting fighters with different weapons and gear. How the game might have looked if GW took a more Necromunda style approach and stuck with those original warbands giving the fighters more options.
